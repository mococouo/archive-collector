## v1.0.0

Initial public release.

### Added

- Archive collection from a source folder into one destination folder.
- GUI and CLI modes.
- Copy or move mode.
- Recursive scanning, with `--flat` for top-level-only scans.
- Duplicate filename handling.
- Default archive extension support for ZIP, RAR, 7z, tar, gzip, bzip2, xz, and zstd formats.
- English default interface.
- Built-in locale files for English, Chinese, French, German, and Japanese.
- `--lang` and `--list-languages` CLI options.
- Cross-platform destination folder opening for Windows, macOS, and Linux.
- README screenshot and GitHub metadata guidance.
